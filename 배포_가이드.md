# 여린교회 앱 Vercel 배포 가이드

## 📦 준비물
이 폴더 안에 있는 파일들:
- index.html
- manifest.json
- icons/ (폴더 전체)
- Code.gs (이건 Vercel에 올리는 거 아님! 구글 앱스 스크립트에 붙여넣을 거)

---

## 1️⃣ 구글 앱스 스크립트 먼저 준비 (5분)

### 1. Code.gs 교체
1. 기존 여린교회 앱스 스크립트 프로젝트 열기
2. `Code.gs` 내용 **전체 선택 → 삭제**
3. 이 폴더의 `Code.gs` 내용 **전체 복사 → 붙여넣기**
4. 저장 (Ctrl+S)

### 2. 새로 배포하기 (중요!)
1. 우측 상단 **"배포"** → **"새 배포"**
2. **유형: 웹 앱**
3. 설명: "Vercel API" 같은 거 입력
4. 다음 2개 설정 확인:
   - **실행: 나** (본인 계정)
   - **액세스: 모든 사용자** ← 이거 꼭 확인!
5. **배포** 클릭
6. **"웹 앱 URL"**이 나옴 → **이 URL을 복사** (이게 제일 중요!)
   - 형태: `https://script.google.com/macros/s/AKfycbx...../exec`

✅ **이 URL을 메모장에 붙여놓기**. 다음 단계에서 씀.

---

## 2️⃣ index.html에 GAS URL 넣기

1. 이 폴더의 `index.html`을 메모장이나 에디터로 열기
2. 맨 위쪽에서 이 줄 찾기:
   ```
   window.GAS_URL = 'YOUR_GAS_URL_HERE';
   ```
3. `YOUR_GAS_URL_HERE`를 방금 복사한 GAS URL로 바꾸기:
   ```
   window.GAS_URL = 'https://script.google.com/macros/s/AKfycbx...../exec';
   ```
4. 저장

---

## 3️⃣ Vercel 가입 (1분)

1. `https://vercel.com` 접속
2. 오른쪽 상단 **"Sign Up"** 클릭
3. **"Continue with Google"** 선택
4. 본인 구글 계정으로 로그인
5. 이름 입력하고 완료 (개인용이니까 Hobby 무료 플랜)

---

## 4️⃣ 파일 업로드 (3분)

### 방법 A: 드래그 앤 드롭 (제일 쉬움)
1. Vercel 로그인 후 대시보드에서 **"Add New..."** → **"Project"**
2. 상단의 **"clone template"** 말고 **"Deploy"** 쪽 보면 업로드 옵션
3. 하단에 있는 **"Browse"** 또는 **파일 드래그 영역**에 이 폴더 통째로 드래그
   (`index.html`, `manifest.json`, `icons` 폴더 다 포함해서)
4. 프로젝트 이름 입력: **`yeorin-ch`** (나중에 `yeorin-ch.vercel.app`이 됨)
5. **Deploy** 클릭
6. 30초~1분 기다리면 완료

### 방법 B: 드래그가 안 보이면
1. "Import Git Repository" 화면이 나올 수 있음
2. 아래로 내리면 **"Deploy a new project without importing"** 또는
   **"Deploy from Template"** 같은 링크가 있음
3. 안 보이면 Vercel CLI라는 방법도 있는데 복잡함 → 내가 도와줄게, 나한테 말해줘

---

## 5️⃣ 완료 후 확인

1. 배포 끝나면 **"Visit"** 버튼 또는 URL 뜸
2. `https://yeorin-ch.vercel.app` 같은 주소로 접속
3. 앱이 기존처럼 잘 뜨는지 확인
4. 로그인해서 기존 데이터(성경진도, 챌린지, QT, 기도제목) 다 그대로 있는지 확인 ✅

---

## 6️⃣ 홈화면에 추가 테스트

### 아이폰 (Safari)
1. `yeorin-ch.vercel.app` 접속
2. 아래 **공유 버튼** 클릭
3. **"홈 화면에 추가"** 선택
4. **여린교회 아이콘**으로 홈화면에 설치됨 ✨

### 안드로이드 (Chrome)
1. `yeorin-ch.vercel.app` 접속
2. 우측 상단 **⋮** 메뉴 → **"홈 화면에 추가"** 또는 **"앱 설치"**
3. **여린교회 아이콘**으로 홈화면에 설치됨 ✨

---

## 🚨 문제 발생 시

### "CORS 오류" 또는 "로그인 안 됨"
→ 2번 단계에서 GAS URL 제대로 넣었는지 확인
→ Code.gs 배포 시 "액세스: 모든 사용자" 체크했는지 확인

### "Code.gs 새로 배포"를 안 하고 저장만 하면 반영 안 됨
→ GAS는 **저장 ≠ 배포**. 꼭 "새 배포"를 눌러야 바뀐 코드가 적용됨!

### 아이콘이 안 바뀜
→ 폰 재부팅 또는 기존 아이콘 삭제 후 다시 추가
→ 캐시 문제일 수 있음

### URL 이름이 이미 사용 중
→ Vercel에서 다른 이름 제안해줌 (예: `yeorin-ch-2024`)
→ 아무거나 괜찮은 걸로 선택하면 됨
